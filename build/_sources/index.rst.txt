MyMCA Documentation
===================

Welcome to the official documentation for the MyMCA project.  MyMCA is a small-scale web-based application with the ability for users to register for YMCA programs.  If you are a user looking for insight on how to operate our software, please refer to the :doc:`UserManual/general`.  If you are a developer, refer to :doc:`Dev/general`.
